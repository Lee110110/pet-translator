# AI宠物翻译官

> 不是宠物百科，是每只宠物的私人健康侦探。

## 这是什么？

AI宠物翻译官是一款帮助你发现宠物健康异常的智能助手。宠物不会说话，生病早期的表现往往只是"食量减少"、"躲起来"这类不起眼的行为变化，主人发现时通常已到中晚期。这款产品的核心思路是——

**不是"翻译宠物说了什么"，而是"翻译宠物身体在说什么"。**

通过每日4项健康指标的快速打卡，系统自动为每只宠物建立个体基线（"这只猫自己的正常值"），一旦偏离基线就主动预警，帮你在问题还没恶化时就发现它。

---

## 核心功能

### 每日健康打卡
每天花15秒记录4项核心指标：饮食量、饮水量、排便状态、精神状态。提交后AI即时给出解读和建议，告诉你"今天有没有问题"。

### 个体基线引擎
连续打卡7天后，系统自动为你的宠物建立个性化基线——不是品种平均值，而是"这只猫自己的正常值"。后续打卡会与基线对比，发现偏离自动预警。基线随年龄、季节动态调整。

### 四级预警系统
- 🟢 **绿色**：一切正常
- 🟡 **黄色**：轻微偏离，建议观察
- 🟠 **橙色**：持续异常或多指标偏离，建议48h内问诊
- 🔴 **红色**：急症信号（如拒食+呕吐+嗜睡），建议立即就医

### 症状对话
发现异常后可启动最多3轮的AI对话，通过结构化问答逐步缩小判断范围，给出明确的行动建议而非模棱两可的"建议就医"。对话中会参考猫咪症状知识库，结合品种和年龄做针对性分析。

### 就医摘要生成
一键生成结构化的就诊摘要，包含：症状时间线、基线偏离数据、预警历史、既往对话评估。可直接复制发给宠物医生，让医生30秒掌握全貌。

### 疫苗/驱虫日历
根据宠物生日自动生成接种计划，到期提醒，一键标记完成。涵盖猫三联疫苗、狂犬疫苗、体内驱虫、体外驱虫。

### 图像智能识别（进阶）
拍摄食物碗、猫砂盆、呕吐物等照片，AI自动识别并给出分析：
- 食物碗 → 判断剩余量、进食评估
- 猫砂盆 → 判断排便形态、排尿状况
- 呕吐物 → 判断呕吐类型、紧急程度

### 疾病概率评估
输入观察到的症状，系统基于品种风险、年龄段和症状组合计算相关疾病概率排序，告诉你最可能是什么问题、需要做哪些检查。

## 技术架构

```
┌───────────────────────────────────────────────────────┐
│                    AI宠物翻译官                          │
├───────────────┬───────────────┬───────────────────────┤
│   数据采集层    │    AI引擎层    │      服务输出层        │
├───────────────┼───────────────┼───────────────────────┤
│ 手动打卡       │ 个体基线建模    │ 健康日记 & 行为解读    │
│ 拍照识别       │ 偏差检测引擎    │ 分级预警推送           │
│ (未来硬件对接)  │ 症状-疾病关联   │ 就医决策助手           │
│               │ 图像体态评估    │ 病史摘要生成           │
│               │ 品种风险模型    │ 情绪解读 & 陪伴        │
└───────────────┴───────────────┴───────────────────────┘
```

### 后端

| 技术 | 用途 |
|------|------|
| FastAPI | Web框架 |
| SQLAlchemy + aiosqlite | 异步ORM + SQLite |
| Pydantic v2 | 数据验证 & 配置 |
| DeepSeek (OpenAI SDK) | LLM对话 & Vision图像分析 |
| python-jose + bcrypt | JWT认证 & 密码加密 |

### 前端

| 技术 | 用途 |
|------|------|
| uni-app (Vue 3) | 跨平台小程序框架 |
| Pinia | 状态管理 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| 目标平台 | 微信小程序 (mp-weixin) |

### 知识库

- `cat_symptoms.json` — 猫咪症状分级知识（呕吐、拒食、饮水量增加等）
- `cat_diseases.json` — 猫咪常见疾病（慢性肾病、脂肪肝、尿路堵塞等）
- `breed_info.json` — 品种健康风险（英短HCM、布偶PKD等）
- `cat_vaccines.json` — 猫咪疫苗/驱虫接种计划

## API 端点

所有端点前缀 `/api/v1`：

| 模块 | 端点 | 说明 |
|------|------|------|
| 认证 | `/auth/register` `/auth/login` `/auth/me` | 注册/登录/个人信息 |
| 宠物 | `/pets` `/pets/{id}` | 宠物CRUD |
| 打卡 | `/pets/{id}/checkins` `/pets/{id}/checkins/latest` | 每日打卡+AI解读 |
| 基线 | `/pets/{id}/baselines` `/pets/{id}/baselines/chart` | 基线数据+对比图 |
| 预警 | `/pets/{id}/warnings` `/pets/{id}/warnings/dashboard` | 预警列表+仪表盘 |
| 对话 | `/pets/{id}/conversations` | 症状对话(最多3轮) |
| 疫苗 | `/pets/{id}/vaccines` `/pets/{id}/vaccines/schedule` | 疫苗管理+自动计划 |
| 上传 | `/pets/{id}/uploads/image` `/pets/{id}/uploads/analyze-image` | 图片上传+分析 |
| 就医 | `/pets/{id}/medical/summary` | 就医摘要生成 |
| 疾病 | `/pets/{id}/warnings/disease-probability` | 疾病概率评估 |

## 快速启动

### 后端

```bash
cd backend

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env，填入 DeepSeek API Key

# 创建必要目录
mkdir -p uploads data

# 启动服务
uvicorn app.main:app --reload
```

启动后访问 `http://localhost:8000/docs` 查看完整API文档。

### 前端

```bash
cd frontend

# 安装依赖
npm install

# 开发模式（H5）
npm run dev:h5

# 构建微信小程序
npm run build:mp-weixin
```

构建后用微信开发者工具导入 `frontend/dist/build/mp-weixin` 目录即可预览。

## 和现有宠物App的区别

| 维度 | 现有宠物App | AI宠物翻译官 |
|------|-----------|------------|
| 参照系 | 品种通用标准 | 这只宠物自己的基线 |
| 工作模式 | 主人主动查 | AI主动发现异常后推送 |
| 核心价值 | 记录工具 | 健康预警+决策支持 |
| 异常发现 | 靠主人自己觉得不对 | AI对比基线自动发现 |
| 就医准备 | 主人临时回忆描述 | 一键生成结构化病史摘要 |

一句话：现有产品是"记录你在乎的"，我们是"发现你没注意到的"。
